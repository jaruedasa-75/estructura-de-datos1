# UrbanStyle Store

UrbanStyle es una tienda web de ejemplo con una presentación moderna, catálogo de prendas, carrito y un cierre de compra sencillo.

## Incluye
- Acceso con usuario de demostración.
- Encabezado, barra lateral y pie de página reutilizables.
- Catálogo de productos con búsqueda y filtro por categoría.
- Carrito con cantidades, eliminación y total actualizado.
- Vista adaptada a móvil y escritorio.
- Contacto visible al final para Instagram, Facebook y correo.

## Cómo abrirlo
1. Descomprime el ZIP.
2. Abre la carpeta en Visual Studio Code.
3. Ejecuta un servidor local, por ejemplo:
   - `python -m http.server 5500`
   - o Live Server.
4. Entra a `http://localhost:5500/login.html`.

## Estructura
- `login.html`: acceso a la tienda.
- `index.html`: página principal.
- `components/`: partes reutilizables.
- `data/`: listado de productos.
- `js/`: lógica de la tienda y del carrito.
- `css/`: estilos generales.
- `assets/images/`: imágenes del catálogo.
