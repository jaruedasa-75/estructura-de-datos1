# Organización del proyecto

La tienda está dividida en varias partes para que sea más fácil de entender y mantener.

## Cómo se organizó
- La parte visual está separada de la lógica.
- Los productos se manejan aparte de la interfaz.
- El encabezado, la barra lateral y el pie se reutilizan en toda la página.
- El catálogo, el carrito y el acceso tienen funciones independientes.

## Qué ventajas da
- El código queda más ordenado.
- Cambiar una sección no obliga a tocar todo lo demás.
- Es más fácil corregir errores y agregar nuevas funciones.
- La página se vuelve más clara para presentar y explicar.

## Resultado
Se obtiene una tienda más limpia, fácil de navegar y preparada para seguir creciendo sin volverse confusa.
